# SGA Lite (Entrega - Prática Python)

Projeto simples de um Sistema de Gestão Acadêmica (SGA Lite) com:
- SQLAlchemy (SQLite local)
- Camada de serviços
- API REST com FastAPI
- CLI com Click
- Testes com pytest

## Requisitos
- Python 3.10+
- pip
